# Video-Game-Sales-Analysis
This is my first project on GitHub 

Project Overview:

Dataset: Video Game Sales from Maven Analytics, consisting of 64,107 rows and 12 columns.

Goal: To analyze how total sales trends by genre (e.g., action, sports, puzzle) affect game popularity and identify top-performing genres.

Data Cleaning & Preparation
Cleanup: Removed irrelevant columns like ‘img’ and ‘last update’ and converted the dataset into a table format.

Standardization: Headers were standardized to proper case, and currency formatting was applied to sales columns.

Missing Values:
Filled 17 missing 'Developer' entries with 'NA'
Replaced missing 'Critic Scores' with the column average (7.23)
Filled missing 'Release Date' values using the cell above
Replaced missing sales data with respective column averages

Key Insights:
Most Popular Genres: Based on total global sales, the most popular genres are Misc, Action and Shooter
Top Performing Genres: The highest performing genres identified are Shooter, Sandbox and Sports
